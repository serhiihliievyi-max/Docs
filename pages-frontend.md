# 📱 Архитектура фронтенда — страницы

## Auth

### `/login` — Login / Register

- Таб "Войти" и таб "Зарегистрироваться"
- Поля: email, пароль
- Регистрация: + имя, фамилия, подтверждение пароля
- После регистрации → редирект на `/profile/setup`
- После логина → редирект на `/search` или `/dashboard`

### `/forgot-password` — Восстановление пароля

> [!info] MVP
> Можно отложить. Для демо достаточно логина.

### `/verify-email` — Подтверждение email

- Страница-заглушка после регистрации
- "Мы отправили письмо на your@email.com"
  В МВП верификация и конфирм и восстановление не вхоидт

## Onboarding / Профиль

### `/profile/setup` — Заполнение профиля

- Имя, фамилия (могут быть заполнены из регистрации)
- Телефон
- Фото профиля (upload)
- Кнопка "Продолжить" → `/search`

### `/profile/documents` — Загрузка документов

- Водительское удостоверение (фото)
- Паспорт (фото)
- Статус верификации (pending / verified)

> [!info] MVP
> В MVP без Загрузки документов.

## Поиск авто (арендатор)

### `/search` — Поиск / Карта

Основная страница поиска.

**Панель фильтров:**

- Дата и время начала аренды (date + time picker)
- Дата и время окончания аренды
- Адрес (город поиска или города)
- Тип топлива (чекбоксы)
- Коробка передач (переключатель)
- Максимальная цена в день

**Результаты:**

- Карточки машин (фото, марка, цена/день, рейтинг, расстояние)
- На карте: маркеры с ценой

### `/cars` — Список результатов

- Grid карточек
- Сортировка: цена ↑↓ / рейтинг

### `/cars/:id` — Детали автомобиля

- Галерея фото (слайдер)
- Заголовок: марка модель, год
- Характеристики: топливо, КПП
- Описание
- Блок "Владелец": аватар, имя, рейтинг, кол-во сдач
- Календарь доступности (read-only) - в МВП может будет без календаря
- Отзывы (список)
- Боковая панель: цена + кнопка "Забронировать"

### `/booking/new?carId=:id` — Booking / Checkout

- Выбранный период (можно поменять)
- Итоговый расчёт:
  - Кол-во дней × цена за день
  - Депозит
  - Итого
- Способ оплаты (в MVP: "Наличными при получении")
- Кнопка "Подтвердить бронирование"

### `/booking/:id/confirmation` — Подтверждение брони

- "Заявка отправлена!"
- Детали брони: машина, даты, сумма
- Статус: ожидает подтверждения владельца
- Контакт владельца (после подтверждения)

## Управление авто (владелец)

### `/dashboard` — My cars dashboard

- Список своих машин
- Для каждой: фото, статус (active/inactive/rented), кол-во броней
- Кнопка "Добавить машину"
- Быстрые действия: редактировать, снять с публикации

### `/cars/new` и `/cars/:id/edit` — Добавление / редактирование авто

**Форма:**

- Марка (выпадающий список или autocomplete)
- Модель
- Год выпуска
- Тип топлива
- Коробка передач
- Цена за день
- Депозит
- Описание
- Адрес / (можно заморочится с геолокацией и гуглмапс)
- Загрузка фото (по одной - в МВП можем делать как base64)

### `/cars/:id/availability` — Календарь доступности

- Полноэкранный календарь
- Можно выбирать диапазоны дат
- Переключатель: "Доступна" / "Заблокирована"
- Тоггл "Всегда доступна"
- Уже забронированные даты (read-only, другой цвет)

### `/bookings/incoming` — Входящие брони

- Список заявок на машины владельца
- Фильтр по статусу: pending / confirmed / all
- Для каждой: кто бронирует, машина, период, сумма
- Кнопки "Подтвердить" / "Отклонить" для pending

## Общие разделы

### `/bookings` — История броней (арендатор)

- Список всех своих броней как арендатора
- Фильтр: активные / завершённые / отменённые
- Для каждой: машина, даты, статус, сумма

### `/messages` — Чат

> [!warning] MVP
> В MVP чат не реализован. Контакт через телефон в профиле владельца.

- Список диалогов (по каждой брони)
- Страница диалога: история сообщений + поле ввода

### `/reviews` — Отзывы

- Форма оставить отзыв (после завершения брони)
- Рейтинг 1–5 звёзд
- Текстовый комментарий

### `/wallet` — Кошелёк / Выплаты (владелец)

> [!info] MVP
> Отложено. В MVP расчёт наличными.

### `/settings` — Настройки

- Смена email / пароля
- Уведомления (вкл/выкл)
- Удаление аккаунта

### `/notifications` — Уведомления

- Список системных уведомлений
- Новая бронь, подтверждение, отмена и т.д.
