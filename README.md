# 🚗 CarSharing P2P — Docs

Документация проекта в формате Obsidian Markdown. Здесь хранится вся база знаний: сущности БД, API эндпоинты, флоу пользователей, MVP скоуп.

---

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/lanaogini/car-rental-app.git
cd car-rental-app
```

### 2. Установить Obsidian

Скачать с [obsidian.md](https://obsidian.md) — доступен на Mac, Windows, Linux.

### 3. Открыть папку как Vault

1. Запустить Obsidian
2. **Open folder as vault** → выбрать папку `docs/` (или корень репо, смотри где лежат `.md` файлы)
3. Всё — все ссылки между страницами (`[[...]]`) будут работать

---

## Как держать доки актуальными

**Перед началом работы — всегда делать pull:**

```bash
git pull origin main
```

**Добавил или изменил информацию — коммить с понятным описанием:**

```bash
git add .
git commit -m "docs: добавил описание эндпоинтов для модуля Bookings"
git push origin main
```

### Правила коммитов

Используем префикс `docs:` и пишем **что именно** добавили или изменили:

| Пример коммита | Когда использовать |
|---|---|
| `docs: добавил таблицу entities MVP` | Новый раздел или страница |
| `docs: обновил API эндпоинты — добавил Reviews` | Изменение существующего |
| `docs: исправил опечатки в user-flow` | Правки и корректуры |
| `docs: добавил ссылки в index` | Мелкие дополнения |
