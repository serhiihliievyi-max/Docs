# 📋 Описание проекта

## Суть продукта

CarSharing P2P — это платформа где частные владельцы автомобилей могут сдавать свои машины в аренду, а другие пользователи — находить и бронировать их на нужный период.

> [!info] Аналоги
> По модели похоже на **Airbnb**, но для машин. Или **OLX / AUTO.RIA** — где один и тот же пользователь может и размещать объявления, и откликаться на чужие. Жёстких ролей нет.

## Ключевые сценарии

### Арендатор хочет взять машину
1. Вводит даты, время, геолокацию
2. Получает список доступных машин
3. Выбирает машину, смотрит фото, описание, отзывы
4. Создаёт бронь
5. Ждёт подтверждения от владельца
6. Получает контакт, забирает машину

### Владелец хочет сдать машину
1. Создаёт карточку машины (марка, модель, год, фото и т.д.)
2. Настраивает доступность (конкретные даты или "всегда доступна")
3. Устанавливает цену за день и депозит
4. Получает заявки на бронирование
5. Подтверждает или отклоняет
6. Передаёт машину и получает оплату

## Роли пользователей

> [!warning] Важное архитектурное решение
> В системе **нет жёстких ролей**. Один пользователь может одновременно быть и владельцем машин, и арендатором — как на OLX. Разграничение идёт через контекст: создал машину → owner этой машины, создал бронь → renter в рамках этой брони.

## Монетизация (будущее)

- Комиссия платформы с каждой сделки (например 10–15%)
- Страховка поездки
- Премиум листинг для владельцев

| Frontend          | React / Next.js                     |
| Backend           | NestJS                              |
| API               | REST API                            |
| База данных       | PostgreSQL (Supabase)               |
| Хранилище файлов  | Supabase Storage                    |
| Аутентификация    | JWT (access + refresh)+Google Login |
| Платёжная система | Stripe (v2, в MVP — наличные p2p)   |
